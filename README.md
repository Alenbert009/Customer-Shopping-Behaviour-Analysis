# Customer-Shopping-Behaviour-Analysis
Customer Shopping Behavior Analysis
Tools & Technologies: Python (Pandas), PostgreSQL, Power BI, SQL, DAX, Matplotlib/Seaborn.

Description (Shortened & Impactful):

Executed an end-to-end analytics project on 3,900 customer transactions to guide strategic business decisions.

Managed data preparation in Python, performing extensive cleaning, imputation (median rating by category) , and advanced Feature Engineering (creating age_group and purchase_frequency_days).



Utilized PostgreSQL (via Python integration ) to solve key business problems, including calculating Revenue by Gender and performing Customer Segmentation (Loyal, New, Returning).



Developed a dynamic Power BI Dashboard with filters and KPIs (e.g., 4K Customers, $59.76 Avg Purchase) , employing DAX formulas (COUNT, MIN, MAX, etc.) for crucial metrics and providing four key business recommendations.
